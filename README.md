# MQTenten
Measequoia4用　点消しプラグイン。リトポ/整形用

### もうちょっと詳しい説明
 - 標準ツールで点を消した場合: 接続するすべての面が削除される
 - MQTenTenで点を消した場合: できるだけ面を残しつつ、点だけ除去
